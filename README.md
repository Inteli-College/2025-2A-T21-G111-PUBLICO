## 🧭 About the Project

**Kavex** is a Service-First, Product-Next B2B startup that automates the issuance and receipt of Electronic Service Invoices (**NFS-e**) for the Brazilian international trade sector. Starting from applied consultancy to learn the domain in depth, the project evolved into a cloud-native, multi-tenant **Service as a Software (SaaSo)** platform that embeds itself in the client's operation and delivers an operational outcome rather than a self-service tool.

The architecture, results, and strategy are documented in the reports above. Client operations referenced in the reports are presented in anonymized form.
